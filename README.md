# Hexlet-js Project

Мой первый проект на Хекслете

Проект для изучения JavaScript с использованием Hexlet.


## Установка зависимостей

Для установки всех необходимых зависимостей проекта используйте команду Makefile:

```bash
make install
```

## Линтинг

Для проверки кода на соответствие стандартам используйте команду:

```bash
make lint
```
Для автоматического исправления ошибок используйте команду:

```bash
make lint-fix
```
